# 全局通用口径

> 本文件存储跨项目共享的通用口径定义（非特定业务相关）。
> 项目特有的业务口径请存放在各项目的 `caliber_memory.md` 中。
> **此文件是永久记忆，不可删除已有条目，只可追加或修正。**

---

## SQL 编写规范口径

### 分区字段优先
- 定义：写 SQL 时，每个子查询中的 WHERE 条件必须**优先限制分区字段**，分区字段过滤放在 WHERE 的第一行
- 来源：用户于 2026-02-24 提供

### 禁止使用 CTE
- 定义：不使用 `WITH ... AS` (CTE) 格式，使用子查询嵌套
- 来源：用户于 2026-02-28 提供

### SR 引擎优先
- 定义：优先使用 SR 引擎执行 SQL，SR 引擎可直接查询 TDW 表
- 来源：用户于 2026-02-28 提供

### 百分比展示格式
- 定义：所有百分比指标统一用小数形式展示（如 0.65 表示 65%），不乘以 100
- 来源：用户于 2026-02-13 提供
